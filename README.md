# Why Laravel Can't Guess Your Factory Relationships

![Why Laravel Can't Guess Your Factory Relationships](assets/poster.jpg)

A short write-up about a common Laravel factory pitfall when a model has **multiple relationships pointing to the same related model**.

The post explains:
- why `for($model)` resolves relationships by **model type**
- how to fix ambiguous factory relationships
- when being explicit is better than relying on conventions
